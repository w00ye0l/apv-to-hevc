S26 APV to iPhone Converter

삼성 S26 APV/apv1 영상을 iPhone용 HEVC HDR 또는 Windows 재생용 H.264 SDR MP4로 변환하는 데스크톱 앱입니다.

Windows 다운로드 macOS 다운로드

버튼은 최신 릴리즈 파일로 바로 연결됩니다.

사용 전 확인

  • FFmpeg 포함
    다운로드 zip에는 FFmpeg와 FFprobe가 함께 들어 있습니다.
  • Windows
    압축을 풀고 .exe 파일을 실행합니다. HEVC 코덱이 없는 PC에서는 앱의 Windows용 H.264 SDR 추가 생성 옵션을 사용하세요.
  • 언어
    앱 안에서 한국어, 영어, 일본어 UI를 선택할 수 있습니다.
  • macOS
    압축을 풀고 .app을 우클릭한 뒤 열기를 선택합니다. Apple Silicon은 Rosetta가 필요할 수 있습니다.
