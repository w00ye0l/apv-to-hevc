S26 APV to iPhone Converter

삼성 S26 APV/apv1 영상을 iPhone, QuickTime, iCloud, 카카오톡에서 재생하기 쉬운 MP4로 변환하는 데스크톱 앱입니다.

Windows 다운로드 macOS 다운로드

버튼은 최신 릴리즈 파일로 바로 연결됩니다.

사용 전 확인

  • FFmpeg 필요
    APV 입력은 FFmpeg 8.0 이상 빌드를 권장합니다.
  • Windows
    압축을 풀고 .exe 파일을 실행합니다.
  • macOS
    압축을 풀고 .app을 우클릭한 뒤 열기를 선택합니다.
